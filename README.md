# NL Browser

An advanced research browser and companion tool for [Nexus Legacy](https://nexuslegacy.space), hosted at [nlbrowser-production.up.railway.app](https://nlbrowser-production.up.railway.app).

## Usage

**Via Chrome Extension (recommended)**

1. Install the [NL Browser extension](https://chromewebstore.google.com/detail/nl-browser/eljpfjmfmbeeddcbjjfhhoeigdjfnlog)
2. Log in to [Nexus Legacy](https://www.nexuslegacy.space/lobby)
3. Click the extension icon → **Open Dashboard**

**Without the extension**

1. Go to [nlbrowser-production.up.railway.app](https://nlbrowser-production.up.railway.app)
2. Log in to [Nexus Legacy](https://www.nexuslegacy.space/lobby)
3. Press `F12` → **Application** → **Cookies** → copy the value of `nexus_token`
4. Click **Provide my own credentials** on the NL Browser login screen and paste your token
