Privacy Policy

Last updated: June 19, 2026

NL Browser is a companion research tool for the browser game Nexus Legacy. This policy explains what data we access and how we handle it.

Using NL Browser without an account

You can use NL Browser without logging in. In this mode the tool loads a public catalogue of game definitions and lets you browse the full research and building trees, build and save plans, and open shared build links — all shown at default ("level 0") values. When you are not logged in, no nexus_token is read, no personal or in-game data is sent to our server, and no usage analytics are recorded. Plans you create are saved only in your own browser's local storage. The sections below describe what additionally happens when you choose to log in.

What data we access

The Chrome extension reads the nexus_token cookie from nexuslegacy.space. This cookie is a session token issued by Nexus Legacy when you log in. It is used solely to authenticate requests to the Nexus Legacy API on your behalf.

How we use your data

  • Your session token is sent to the NL Browser server and stored in a temporary server-side session.
  • The token is used to fetch your in-game data from the Nexus Legacy API — including research, buildings, planets, fleet and mission status, reports (survey, spy, expedition, mining, pirate), outposts, directives, wormholes, and market data.
  • If you use the Export feature, these are bundled into a single JSON snapshot that is downloaded directly to your device. NL Browser does not retain a copy of your export.
  • We do not share your token or any personal data with third parties.
  • We do not sell, rent, or trade your data.
  • Your session expires after 7 days or when you log out.

Optional usage analytics

On first login you will be asked whether to allow anonymous usage analytics. If you accept, NL Browser records:

  • That you logged in on a given day (used for DAU/MAU counts — always recorded regardless of consent)
  • Which view mode you use (cards, columns, table)
  • Whether you use the Research Planner
  • Approximate session duration and whether you immediately left

No game data, chat, or personal information is included in any of the above. If you decline, none of the optional analytics are recorded. You can change your preference by clearing your browser's local storage for this site.

Suggestions

If you submit a suggestion via the suggestion box, the text is stored along with your in-game username (if you accepted analytics) or anonymously (if you declined). Suggestions are never shared publicly.

Shared build links

When you create a share link for a plan, that plan — the research and building selections together with the plan name you chose — is stored in our Upstash Redis database and given a short link. Anyone who has the link can view the plan, so a shared link is effectively public. Shared plans are kept indefinitely (they are not automatically deleted) so that existing links keep working. No in-game data, account information, or token is included — only the plan's contents and name. Because the plan name is visible to anyone with the link, please avoid putting personal or sensitive information in plan names. Creating a share link requires being logged in; viewing one does not.

Data storage

Session data is held in memory and discarded when you log out or the session expires. Anonymous usage counts are stored in a private Upstash Redis database and automatically deleted after 35 days. Shared plans are stored in the same database and kept indefinitely so their links remain valid. As with most websites, our server and hosting provider also keep standard request logs (such as IP address and browser type) for security and reliability; these are not used for advertising or cross-site tracking.

Third-party services

NL Browser communicates with nexuslegacy.space APIs to fetch your game data, and uses Upstash Redis to store anonymous usage counts and shared plans. No advertising or tracking services are used.

Your rights

You can log out at any time via the extension popup or the app, which immediately destroys your session. You can revoke the extension's access by uninstalling it from your browser. To opt out of analytics, click Decline on the consent banner (clear local storage to see it again).
